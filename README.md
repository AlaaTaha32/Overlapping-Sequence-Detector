# Overlapping Sequence Detector
A Verilog project that detect and count the number of overlapping "1011" sequences stored in a ROM.

Verilog Modules:

1- 16 * 32 ROM

2- Parallel-in-serial-out (PISO)

3- Finite-state machine

4- Counter

5- Odd checker

6- Testbench
